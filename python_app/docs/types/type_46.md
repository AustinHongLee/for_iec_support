# Type 46 — 曲面設備直接支撐（含 D-80 接口）(Vessel Direct Support with D-80 Detail)

> **圖號**: 46 &nbsp;|&nbsp; **分類**: 支撐型式 &nbsp;|&nbsp; **狀態**: ✅ 已實作
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **圖面頁**: D-56 (1/1)
> **日期**: 12/12/19

**無 Trunnion** 的 Vessel 直接支撐，管線端連接細節引用 **D-80**。
A 型 (θ=30°) / B 型 (θ=45°)。適用 **2"~14"** 管徑（比 Type 44/45 更廣）。
公式 `H = P − √(R²−Q²)` 與 Type 44 相同（無保溫/Lug 偏移修正）。

> **D-80 = Type 66（管鞍座 Pipe Shoe）**: 本圖標註 "DETAIL SEE D-80 (TYP.)"——
> D-80 已確認為 **Type 66 管鞍座**（120° 鞍座，管線↔支撐介面層）。
> 詳見 [type_66.md](type_66.md)。

---

## 系統定位

```
TYPE-44 → 直接承托，無 clamp，8"~14"
TYPE-45 → 直接 + Lug Plate clamp + 保溫修正，8"~14"
TYPE-46 → 直接 + D-80 接口，2"~14" ← 本型
TYPE-42 → Trunnion 承托，2"~24"
TYPE-43 → Trunnion + Lug Plate clamp，2"~24"
```

> Type 46 = Type 44 骨架 + D-80 管線接口 + 管徑範圍擴展到 2"。
> 管線端約束行為（固定/導向/自由）取決於 D-80 細節。

## 幾何公式

### H 計算

```
H = P - √(R² - Q²)
```

| 符號 | 意義 |
|:---:|------|
| P | Vessel ℄ 到管線支撐點水平距離 |
| R | Vessel 半徑 |
| Q | 管線水平偏移（由表 1 查） |

> 與 Type 44 公式完全相同，無偏移修正項。

### 斜撐幾何

| θ | 垂直投影 | 水平基準 | L50 斜撐長 | 條件 |
|:---:|:---:|:---:|:---:|------|
| 30° (A) | 450 | 780 | 1016 | H > 1200 ONLY |
| 45° (B) | 780 | 780 | 1203 | H > 1200 ONLY |

> 與 Type 44 完全相同（780 水平跨距, 同斜撐長, 同安裝條件）。

## 設計流程

```
1. 確定: LINE SIZE (2"~14"), Vessel R, P
2. 查表 1: LINE SIZE → Q
3. 計算 H = P - √(R² - Q²)
4. 選 Channel Member (依管徑+H, 查圖表)
5. 選 θ (30° 或 45°)
6. 判斷斜撐: H > 1200 → 裝 L50×50×6 (1016 or 1203mm)
             H ≤ 1200 → 不裝斜撐
7. 查 D-80 → 管線端連接細節
8. 編碼: 46-{N}-{M}-{H} {A|B}[-{Q}]
```
