# Type 48 — Drain Hub 偏移底座支撐 (Offset Base Support for Drain Hub)

> **圖號**: 48 &nbsp;|&nbsp; **分類**: 支撐型式 &nbsp;|&nbsp; **狀態**: 🔲 參考
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **圖面頁**: D-59 (1/1)
> **日期**: 12/12/19

**最簡支撐之一**——管線正下方有 Drain Hub 或其他障礙物時，以 100mm 偏移量的 L 型底板（cantilever plate）
提供垂直承托。無 clamp、無 guide、無 bolt、無 trunnion、無斜撐。
僅 1/2"~6"（小管徑），板尺寸固定 150×100，厚度依管徑分 6mm / 9mm。

---

## 系統定位

```
TYPE-48 = 偏移底座支撐（幾何避讓型）
用途: 管線正下方有 Drain Hub / 設備出口 / 障礙物時的替代方案
```

### 與 Vessel 支撐家族的區別

Type 48 不屬於 Vessel 支撐家族 (42~47)。它是落地型（GRADE），用於排水管周邊的小管支撐。

---

## 結構組成

| 元件 | 說明 |
|------|------|
| PLATE SIZE "C" | L 型底板，150×100×6 或 150×100×9（依管徑） |
| 6mm 焊接 | Pipe 到 Plate 的連接 |
| 底座 | 落地（GRADE），高度 20mm |

> **整個支撐只有一塊 plate + 焊接**。沒有其他結構件。

---

## 幾何

| 項目 | 值 |
|------|:---:|
| 水平偏移（Pipe ℄ → Plate 中心） | **100 mm** |
| 底座高度 | **20 mm** |
| 包覆角 | 無（plate 直接承托） |
| 焊接 | 6mm |

```
   ─── Pipe ───
       │
       │ 6mm weld
       │
   ┌───┴───┐
   │ PLATE │── 100mm offset
   │  "C"  │
   └───┬───┘
       │ 20
  ═════╧═════ GRADE
     DRAIN HUB
```

> 偏移存在原因：pipe 正下方是 **DRAIN HUB**，不能放支撐。
> Plate 以 cantilever 方式承載 → 偏移產生 **彎矩 = 荷重 × 100mm**。
> 板厚從 6→9mm 時，抗彎剛性提升 (9/6)³ ≈ 3.4 倍。

### 規律

- 全部 plate 平面尺寸一致：**150 × 100 mm**
- ≤ 2"：板厚 **6 mm**
- 3"~6"：板厚 **9 mm**

> 不靠放大尺寸，靠增加板厚抗彎。設計極簡。

## 設計流程

```
1. 確認管線正下方有 Drain Hub 或障礙物
2. 確定 LINE SIZE (1/2"~6")
3. 查表 1: LINE SIZE → PLATE SIZE "C"
4. 查 TABLE "B": 材質 → 符號
5. 編碼: 48-{A}{mat_sym}
```

## 關聯 Type

| 關聯 Type | 關係 |
|------|------|
| Type 44~47 | 完全不同系統——44~47 是 Vessel 斜撐，48 是落地偏移底座 |
| Type 66 (D-80) | 都是管線介面件，但 66 是鞍座 (120°包覆)，48 是 plate (平面承托) |

### Type 48 vs 其他小型支撐

| 特徵 | Type 48 |
|------|---------|
| 位置 | 落地 (GRADE) |
| 管徑 | 1/2"~6"（小管專用） |
| 結構 | 單一 plate |
| 偏移 | 100mm（避開 Drain Hub） |
| 約束 | 僅垂直 |
| 複雜度 | 最低（2 項 BOM） |
