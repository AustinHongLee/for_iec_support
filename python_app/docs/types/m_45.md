# M-45 — Expansion Bolt（膨脹螺栓）

> **圖號**: M-45 &nbsp;|&nbsp; **分類**: 零件 (Component) &nbsp;|&nbsp; **狀態**: 🔲 參考
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **日期**: 12/12/19

混凝土用膨脹螺栓（Cinch Bolt），用於將 Base Plate 錨定至 R.C. 結構。
主要搭配 Type 41（牆面錨定支撐）及 M-42（底板，grout 25mm）。

---

## 結構示意圖

```
          ┌─ D ─┐
          │     │
    ══════╪═════╪══════  ← TH'K OF FIXED PLATE
          │ NUT │
          │STEEL│
          │WASHER
    ──────┼─────┼──────  ← 混凝土表面
          │     │
          │ ↕ L │       THREAD
          │     │       LENGTH
          │  ╲╱ │       ↕
          │  ╱╲ │  ← 膨脹錨定段
          └─────┘
    ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  R.C. CONCRETE
```

**力傳遞**：Base Plate → Nut/Washer → Bolt 桿 → 膨脹段 → 混凝土摩擦/擠壓

---

## 本質

> **Cinch Bolt = 標準膨脹螺栓**，靠膨脹段與 R.C. 孔壁的摩擦力傳遞拉力與剪力。

---

## 編碼格式

**格式**: `EB-{D}`

| 段位 | 意義 |
|:---:|------|
| EB | Expansion Bolt |
| D | 螺栓直徑（英寸） |

**範例**: `EB-1/2`, `EB-3/4`

## 設計條件

| 參數 | 值 | 說明 |
|------|:---:|------|
| R.C. 強度 | 210 kg/cm² | 混凝土抗壓強度 |
| 安全係數 | 5 | NOTE 3 明確規定 |

> **容許載荷 = 極限載荷 / 5**

---

## 選用規則

Type 41 表中 EXP. BOLT DIA 直接對應：

| Type 41 Bolt DIA | M-45 型號 |
|:---:|:---:|
| 1/2" | EB-1/2 |
| 5/8" | EB-5/8 |
| 3/4" | EB-3/4 |

> Type 41 只用到 1/2"~3/4" 三種；EB-1/4, EB-3/8, EB-7/8 供其他用途。

---

## 工程 NOTES

1. **DESIGNATION**: `EB-{D}` — D = 螺栓直徑 (inch)
2. **用途限制**: These standard cinch bolts shall be used **only to small pipe supports**
3. **載荷基準**: R.C. strength 210 kg/cm²，safety factor = 5
4. **GROUT**: M-42 底板型式中，TYPE B/E/G/L/M 需灌 25mm grout

---

## 關聯 Type

| 關聯 Type | 用途 |
|------|------|
| Type 41 | 牆面錨定支撐，Base Plate + Expansion Bolt 固定至混凝土 |
| M-42 (底板) | 部分型式需 grout 25mm + Expansion Bolt |

> Expansion Bolt 為承力構件——拉力 + 剪力同時作用時需進行組合載荷校核。
