## 系統本質

立柱式滑動支撐+機械止擋(Stopper)，Channel N + 黑鐵立柱 + M42(僅G/J)

立柱式托板支撐，管線坐在上板可滑動，旁邊Stopper限制最大位移。Channel N(C100/C125)橫向支撐，Pipe A立柱(黑鐵)。H/L現場裁切，H≤1500mm，L≤1000mm。M42僅G/J。
