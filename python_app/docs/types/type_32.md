## 系統本質

從EXISTING STEEL向下吊掛的U型框, 無M-42

上部焊接EXISTING STEEL, 管線在框內。與Type 31鏡像: 31=由下往上撐, 32=從上往下吊。現行 BOM 拆為左腿+底梁+右腿三件。
