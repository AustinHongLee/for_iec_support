# Type 61 — Trunnion 支座設計 (Trunnion Design for Pipe Support)

> **圖號**: 61 &nbsp;|&nbsp; **分類**: 支撐型式 (接口零件) &nbsp;|&nbsp; **狀態**: ✅ 已實作
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **圖面頁**: D-72 (1/3), D-73 (2/3), D-74 (3/3)
> **日期**: 12/12/19

Trunnion（短管支座）的構造、焊接、尺寸與力矩容量完整定義。
**三頁一組**：D-72 = 構造/焊接/編碼，D-73 = 碳鋼容量表，D-74 = 不鏽鋼容量表。
Type 42 / Type 43 的 Trunnion 結構強度由此驗算。

---

## 系統定位

```
管線 → Trunnion (Type 61) → Vessel
           ↓
       支撐結構 (Type 42 / 43)
```

> Trunnion 是**主要載荷傳遞元件**。Type 42/43 的圖面只定義幾何佈局，
> 結構是否成立由 D-73/D-74 的力矩容量表決定。

## D-72：構造細節（第 1 頁）

### Trunnion 型式

| 型式 | 符號 | 說明 |
|:---:|:---:|------|
| 單管 | T1 | 單支 Trunnion |
| 雙管 | T2 | 雙支 Trunnion，力矩容量加倍（NOTE 3） |

### 安裝型式

| 型式 | 後綴 | 說明 |
|:---:|:---:|------|
| 無補強墊板 | — | 直接焊在 pipe 上，H = 管壁到端部 |
| 有補強墊板 | (P) | 加 reinforcing pad，H 含 pad，有 Ø6 weep hole |
| 穿管型 | HP | pad + 穿 pipe（但 NOTE: DO NOT CUT HOLE IN PIPE） |

### 焊接細節

| 條件 | 焊道 | 說明 |
|------|------|------|
| NO PAD | 6mm fillet, 45° | ≤4" 用 6mm，>4" 用 9mm |
| PAD REQ'D | 6mm fillet + E/F 尺寸 | F = 6mm (alloy/stainless), 9mm (carbon steel)（NOTE 4） |

### 關鍵尺寸

- 100mm：Trunnion 管頂到上方結構間距
- 10 MIN.：Trunnion 端面到 pipe 邊緣最小距離
- 12(TYP.)：焊道標準尺寸
- C/S SHIM IN FIELD (TYP.)：現場調整墊片

## 編碼格式

**格式**: `61-{PIPE_SIZE}-T{n}-{LENGTH}[(P)]`

| 段位 | 意義 | 說明 |
|:---:|------|------|
| 1 | Type 編號 | 固定 `61` |
| 2 | Trunnion 管徑 | NOM. TRUNN. PIPE SIZE（如 3B） |
| 3 | 型式 | T1 = 單管, T2 = 雙管 |
| 4 | 長度 | Trunnion 長度，以 100mm 為單位（如 05 = 500mm） |
| 5 | (P) | 可選，表示需要 reinforcing pad |

**範例**: `61-3B-T1-05(P)` = Type 61, 3" trunnion, 單管, 500mm, 需 pad

## 設計驗算流程

```
1. 確定管線: NOM. PIPE SIZE, SCH, MATERIAL (CS / SS)
2. 選 Trunnion size (由 Type 42 表 2 或工程需求)
3. 查力矩容量表:
   - 碳鋼 → D-73
   - 不鏽鋼 → D-74 (依 TP304/316/347 選修正係數)
4. 溫度修正: 容許力矩 = 表查值 ÷ 溫度修正係數
5. 判斷 PAD 需求:
   - NO PAD 容量 ≥ 設計力矩 → 不需 pad
   - NO PAD 不夠 → 加 PAD (查 pad TH'K 和 DIM D)
   - PAD 仍不夠 → 改用 T2 雙管 (容量 ×2)
6. 確認焊接細節 (D-72)
```

---

## 工程 NOTES（原圖標註）

1. **DESIGNATION**: `61-{B}-T{1|2}-{L/100}[(P)]`
2. **材質規定**: 合金鋼、不鏽鋼、需消除應力的管線——Trunnion 材質必須與主管相同（NOTE 2）
3. **雙管容量**: T2 = T1 × 2（NOTE 3）
4. **焊接 F 尺寸**: Alloy/Stainless = 6mm, Carbon steel = 9mm（NOTE 4）
5. **不可在 pipe 上開孔**: DO NOT CUT HOLE IN PIPE
