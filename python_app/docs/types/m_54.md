## Unit Weight

M-54 圖面沒有 unit-weight 欄位。目前 Fig.2 使用 strap blank 幾何計算：

```text
weight_kg = (B*C - 2*pi*(11/2)^2) * T * 7.85e-6
```

這代表重量是 table-backed calculated value，不是 source drawing 直接列值。
