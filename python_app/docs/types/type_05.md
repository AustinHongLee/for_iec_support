## 系統本質

≤2" 角鋼立柱支撐，U-bolt固定(不含供貨)，M42僅允許 D/L/P/R

適用於2"及以下管線的立柱式上托支撐。角鋼Member M(L50/L65/L75)作垂直柱+水平托板(130mm)，U-bolt束制管線(不含供貨,見D-68)。H現場裁切,最大1500mm。需搭配M-42下部構件,僅允許D/L/P/R。
