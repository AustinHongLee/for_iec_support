## 系統本質

彎頭滑動支撐，含dummy pipe(B) + 支撐柱(C) + 底板(E) + 滑動板(F) + M42(僅J)

有滑動面的彎頭支撐，允許管線熱位移。Pipe B(dummy)=L+100，Pipe C(支撐柱)=H-100-板厚。H現場裁切(2000~3500mm)。M42僅允許J型。特殊材質同主管。
