## 系統本質

以螺牙桿+假管調整高度的支撐型式，螺桿壓在底板上，非焊接固定。

Type 09 使用 2" dummy pipe 焊在主管上作為支撐腿，底端焊接螺帽，再以 1-5/8" 全牙螺桿穿過螺帽調整高度，螺桿底部壓在 M-42 底板上。適用 2"~4" 管徑，H≤1500mm。
