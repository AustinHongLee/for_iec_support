# Type 31 — 框架式支撐 (Frame Support on Existing Steel)

> **圖號**: 31 &nbsp;|&nbsp; **分類**: 支撐型式 &nbsp;|&nbsp; **狀態**: ✅ 已實作

焊接於既有鋼構上的矩形封閉框，以雙柱 + 上橫梁形成剛性框架承載管線。

---

## 結構示意圖

```
          ┌──────── L ────────┐
          │                   │
     ═════╪═══════════════════╪═════  ← 上橫梁 (BEAM)
          │                   │
          │  H    ☀ PIPE  H   │       ← 雙柱 (COLUMN ×2)
          │                   │
     ─────┴───────────────────┴─────  ← EXISTING STEEL (底部)
```

**力傳遞**：管線重量 → 上橫梁(L) → 雙柱(H×2) → 既有鋼構

---

## 編碼格式

**格式**: `31-{MEMBER}-{LLHH}`

| 段位 | 意義 | 說明 |
|:---:|------|------|
| 1 | Type 編號 | 固定 `31` |
| 2 | 型鋼代碼 | L50, L75, C125, C150, H250 |
| 3 | LLHH | 前2位=L×100mm, 後2位=H×100mm |

**範例**: `31-L50-1005` → L50角鐵, L=1000mm, H=500mm
