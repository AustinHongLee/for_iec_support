# Type 44 — 曲面設備直接斜撐支撐 (Vessel Direct Inclined Support)

> **圖號**: 44 &nbsp;|&nbsp; **分類**: 支撐型式 &nbsp;|&nbsp; **狀態**: ✅ 已實作
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **圖面頁**: D-53 (1/1)
> **日期**: 12/12/19

**無 Trunnion、無 Lug Plate、無 Clamp** 的簡化 Vessel 支撐——管線直接靠在 Channel 上承托。
A 型 (θ=30°) / B 型 (θ=45°)。僅適用 8"~14" 管徑。
是 Type 42/43 系列中載荷最輕、結構最簡的型式。

---

## 系統定位

```
TYPE-37 → 平面斜撐（牆面/鋼架）
TYPE-39 → 曲面 + Lug Plate 直接接 Vessel
TYPE-42 → 曲面 + Trunnion 承托
TYPE-43 → 曲面 + Trunnion + Lug Plate clamp（全約束）
TYPE-44 → 曲面 + Channel 直接承托（最簡化） ← 本型
```

> 無 Trunnion → 不需 D-72/73/74 校核。
> 無 Clamp → 管線僅「靠」在 Channel 上，非固定約束。
> 適用：中等載荷、無需全約束、結構從簡的 Vessel 管線支撐。

- Type 42/43: Pipe → Trunnion → Vessel → 需 D-72/73/74 校核

## 幾何公式

### H 計算（NOTE 2）

```
H = P - √(R² - Q²)
```

| 符號 | 意義 |
|:---:|------|
| P | Vessel ℄ 到管線支撐點水平距離 |
| R | Vessel 半徑 |
| Q | 管線水平偏移（由表 1 查） |

### 與 Type 42/43 公式比較

| | Type 42 | Type 43 | Type 44 |
|------|---------|---------|---------|
| 公式 | H = F − √(R²−E²) | H = K − √(R²−Q²) − t − E − 30 | H = P − √(R²−Q²) |
| 保溫扣除 | 未明確 | 扣 t | 無 |
| Lug 偏移 | 無 | 扣 E + 30 | 無 |
| Trunnion 校核 | 需要 | 需要 | **不需要** |

> Type 44 公式最簡——只有曲面幾何，無任何零件偏移修正。

### 斜撐幾何

| θ | 垂直投影 | 水平基準 | L50 斜撐長 | 條件 |
|:---:|:---:|:---:|:---:|------|
| 30° (A) | 450 | 780 | 1016 | H ≥ 1200 ONLY |
| 45° (B) | 780 | 780 | 1203 | H ≥ 1200 ONLY |

> 780 為固定水平跨距。
> 驗算：tan(30°) = 450/780 = 0.577 ✓，tan(45°) = 780/780 = 1.0 ✓

## 設計流程

```
1. 確定: LINE SIZE (8"/10"/12"/14"), Vessel R, P
2. 查表 1: LINE SIZE → Q
3. 計算 H = P - √(R² - Q²)
4. 選 Channel Member (依管徑+H, 查圖表)
5. 選 θ (30° 或 45°)
6. 判斷斜撐: H ≥ 1200 → 裝 L50×50×6 (1016 or 1203mm)
            H < 1200 → 不裝斜撐
7. 編碼: 44-{N}-{M}-{H} {A|B}[-{Q}]
```
