# Type 47 — 曲面設備直接夾持支撐（含 D-80 接口）(Vessel Direct Clamp Support with D-80 Detail)

> **圖號**: 47 &nbsp;|&nbsp; **分類**: 支撐型式 &nbsp;|&nbsp; **狀態**: ✅ 已實作
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **圖面頁**: D-57 (1/2), D-58 (2/2)
> **日期**: 12/12/19

**無 Trunnion，有 Lug Plate clamp + D-80 接口** 的 Vessel 直接支撐。
A 型 (θ=30°) / B 型 (θ=45°)。適用 **2"~14"** 管徑。
公式 `H = P − √(R²−Q²) − 60 − t` 與 Type 45 完全相同。

> **本型 = Type 45（Lug Plate clamp 系統）+ Type 46（D-80 接口 + 2"~14" 管徑範圍 + 大 Q 值）**

> **D-80 = Type 66（管鞍座 Pipe Shoe）**: 本圖標註 "DETAIL SEE D-80 (TYP.)"——
> D-80 已確認為 **Type 66 管鞍座**（120° 鞍座）。本型同時有 Shoe + Lug Plate。
> 詳見 [type_66.md](type_66.md)。

---

## 系統定位

```
TYPE-44 → 直接承托，無 clamp，8"~14"
TYPE-45 → 直接 + Lug Plate clamp + 保溫修正，8"~14"
TYPE-46 → 直接 + D-80 接口，2"~14"（無 Lug Plate）
TYPE-47 → 直接 + Lug Plate clamp + D-80 接口，2"~14" ← 本型
TYPE-42 → Trunnion 承托，2"~24"
TYPE-43 → Trunnion + Lug Plate clamp，2"~24"
```

> Type 47 是 Type 45 + Type 46 的複合型：
> - 從 Type 45 繼承：Lug Plate (M-34 + M-35/36) + H 公式 (-60-t) + 斜撐條件 (H>1140)
> - 從 Type 46 繼承：D-80 接口 + 管徑 2"~14" + Q 值 (133~281mm)

## 幾何公式

### H 計算

```
H = P - √(R² - Q²) - 60 - t
```

| 符號 | 意義 |
|:---:|------|
| P | Vessel ℄ 到管線支撐點水平距離 |
| R | Vessel 半徑 |
| Q | 管線水平偏移（由表 1 查） |
| t | 保溫層厚度 (INSUL. TH'K) |
| 60 | Lug Plate 固定偏移量 (mm) |

> 與 Type 45 公式完全相同（含 -60 和 -t 修正）。
> 與 Type 44/46 不同（44/46 沒有 -60 和 -t）。

### 斜撐幾何

| θ | 垂直投影 | 水平基準 | L50 斜撐長 | 條件 |
|:---:|:---:|:---:|:---:|------|
| 30° (A) | 400 | 758 | 1008 | H > 1140 ONLY |
| 45° (B) | 669 | 758 | 1150 | H > 1140 ONLY |

> 與 Type 45 完全相同（758 水平跨距, 1008/1150 斜撐長, H>1140 條件）。
> 與 Type 44/46 不同（44/46 用 780 水平跨距, 1016/1203 斜撐長, H>1200 條件）。

## 設計流程

```
1. 確定: LINE SIZE (2"~14"), Vessel R, P, 保溫厚 t
2. 查表 1: LINE SIZE → Q
3. 計算 H = P - √(R² - Q²) - 60 - t
4. 選 Channel Member (依管徑+H, 查圖表)
5. 查表 2: Member → A/B/C/D/E/F/G/J/K 尺寸
6. 選 θ (30° 或 45°)
7. 判斷斜撐: H > 1140 → 裝 L50×50×6 (1008 or 1150mm)
             H ≤ 1140 → 不裝斜撐
8. 配置 Detail Y: M-35/36 (Vessel端 Lug Plate D/E)
9. 配置 Detail Z: M-34 (管線端 Lug Plate C)
10. 查 D-80 → 額外管線端接口
11. 編碼: 47-{N}-{M}-{H} {A|B}[-{Q}]
```
