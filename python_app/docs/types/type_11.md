## 系統本質

利用彈簧承受管重與位移變化的可變載支撐，dummy pipe與全牙螺桿傳力，spring為主功能元件。

Type 11 透過 1.5" SCH.80 dummy pipe 將主管載重傳至 1-5/8" 全牙螺桿，再經由彈簧(SPR12/SPR14)吸收位移，下方為 2" SCH.40 支撐管柱。彈簧須現場預壓設定(preset in field)。
