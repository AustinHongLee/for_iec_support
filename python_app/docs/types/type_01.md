## 系統本質

以焊接假管 (Dummy Pipe) 從主管線延伸至底板的基本支撐型式，適用 2"~50" 管線。

Type 01 是最常見的管線支撐型式。上段管材質跟隨主管線（如 SUS304），下段管固定使用 A53Gr.B 黑鐵管。上段管長度由 L（查表，基於 Long Radius Elbow）+ 100mm 決定，下段管長度為 H×100 - 100mm。底板由第三段字母代碼（A~T）決定，對應 M-42 圖面。H ≤ 1500mm，支撐到地面或鋼板 (TO GRADE OR PLATE)。

---

## 編碼格式

```text
01-{A}B-{H}{M42字母}
```

---

## 核心運算邏輯

```text
上段管長 = L + 100mm（Elbow 查表）或 100 + M（Tee 從 ASME B16.9 查表）
下段管長 = H×100 - 100mm
上段材質 = 跟隨主管線
下段材質 = A53Gr.B（固定）
底板 = M-42 依字母代碼
```

---

## 設計重點

- 與 `M-42` 下部構件有關，最終組成會受到末段字母或允許型別限制。
