# M-36 — Lug Plate TYPE-E（60° 補償轉接板）

> **圖號**: M-36 &nbsp;|&nbsp; **分類**: 零件 (Component) &nbsp;|&nbsp; **狀態**: 🔲 參考
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **日期**: 12/12/19

60° 角度 Lug Plate，帶 `0.577T` 幾何補償，用於 θ=30° 斜撐與 Vessel/結構間的精準載荷轉接。
主要搭配 θ=30° 支撐系統（如 Type 39 FIG-A 的 DETAIL Z）。

---

## 結構示意圖

```
         ┌──── A ────┐
    t    │            │
  ──┤    │   ╱ C ╲    │
    │    │ ╱ D ╲╱ D╲  │
    │    │╱  ⊙  ⊙  ╲  │──── MEMBER
    B    │ ╲ E ╱╲ E╱  │
    │    │   ╲  ╱  ╱   │
    │    │    ╲╱  ╱    │
    │   60°   ╲ ╱     │  ← 板角度 60°
  ──┤    │     ╲      │
  0.577T │  ØJ HOLE   │  "K" BOLT
  ───┤   │  FOR BOLT  │
    T    │            │
  PLATE  └─ "T" TH'K ─┘
         INSULATION
```

**力傳遞**：PIPE → INSULATION → LUG PLATE (60°) → K BOLT → MEMBER

## 本質與定位

Lug Plate 家族共三型：

| 型號 | 角度 | 用途 |
|:---:|:---:|------|
| TYPE-C (M-34) | 0° (平面) | 水平接、一般固定 |
| TYPE-D (M-35) | 45° | 一般斜接轉換 |
| **TYPE-E (M-36)** | **60°** | **30° 角度補償（含 0.577T 修正）** |

### 為什麼需要 TYPE-E？

θ=30° 的斜撐系統中：
- Member 是直的
- Pipe 是水平的
- 中間需要一塊**幾何轉換板**，將 30° 斜撐角度轉成 bolt 平面

> **`0.577T`** = `tan(30°) × T`，圖面直接標註，用於精準補償板厚在斜面的投影。

TYPE-E 比 TYPE-D 更「數學化」——不只是轉角，還修正板厚幾何偏差。

## TYPE-D vs TYPE-E 尺寸差異

以同一 Member 比較，TYPE-E (60°) 相對 TYPE-D (45°) 的差異：

| 特徵 | TYPE-D (M-35) | TYPE-E (M-36) | 趨勢 |
|------|:---:|:---:|------|
| A (板寬) | 較小 | 較大 | E 多 15~30mm |
| B (板高) | 較大 | 較小 | D 多 40~200mm |
| 幾何補償 | 無 | 0.577T | E 有角度修正 |

> 原因：60° 板更「矮胖」，45° 板更「高瘦」。

### θ 角度與 Lug Plate 選用

| 斜撐角度 θ | Lug Plate | 理由 |
|:---:|:---:|------|
| 0° (無斜撐) | TYPE-C (M-34) | 平面接合 |
| 30° | **TYPE-E (M-36)** | 60° 板 + 0.577T 補償 |
| 45° | TYPE-D (M-35) | 45° 板直接轉接 |

> ❌ TYPE-D 與 TYPE-E **不可互換**——幾何偏差會影響 bolt alignment 與載荷路徑。

## 工程 NOTES

1. **DESIGNATION**: `LGP-E-{n}` — E 表示 TYPE NO.，LGP 表示 LUG PLATE
2. **MATERIAL**: Same as the metal to which it is connected（與連接構件同材質）
3. **REINFORCING PAD**: 若加補強墊板，PAD 厚度已含在 A 或 A+t 內
