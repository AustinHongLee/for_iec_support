## 系統本質

角鋼/H型鋼斜撐，限制管線側向位移

以 L-angle 或 cut-from H-beam 作為斜撐，焊接於主管與支撐柱間形成三角穩定結構。1~6吋用角鋼(L40/L50/L75)，8~12吋用 CUT FROM H194X150X6X9。L 為現場裁切。
