# Type 42 — Trunnion 曲面設備斜撐支撐 (Trunnion Vessel Braced Support)

> **圖號**: 42 &nbsp;|&nbsp; **分類**: 支撐型式 &nbsp;|&nbsp; **狀態**: ✅ 已實作
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **圖面頁**: D-50 &nbsp;|&nbsp; **日期**: 12/12/19

透過 Trunnion（短管支座）將管線支撐轉嫁到 Vessel 的參數化斜撐系統。
A 型 (θ=30°) 斜撐較平緩，B 型 (θ=45°) 斜撐較陡。
**比 Type 39 更進階**——不直接夾管，而是透過 Trunnion 分散應力、適用高溫/厚保溫場景。

## 核心幾何關係

### H 的計算（曲面幾何反算）

```
H = F - √(R² - E²)
```

| 變數 | 意義 | 單位 |
|:---:|------|:---:|
| F | Vessel 中心到支撐外側距離 | mm |
| R | Vessel 半徑 | mm |
| E | 管中心到 Vessel 中心線偏移量 | mm |
| H | 反算後的斜撐高度 | mm |

> **比 Type 39 更進一步**：直接用圓弧幾何計算，考慮 Vessel curvature。
> Type 39: `H = K - R - t - E - 30`（線性近似）
> Type 42: `H = F - √(R² - E²)`（精確圓弧）

### θ 角度選擇（NOTE 4）

| 代號 | θ | 說明 |
|:---:|:---:|------|
| A | 30° | 斜撐較平緩 |
| B | 45° | 斜撐較陡 |

### G 的計算（斜撐長度）

| θ | G 公式 | 係數來源 |
|:---:|------|------|
| 30° | `1.155H + offset` | 1/cos(30°) ≈ 1.1547 |
| 45° | `1.414H + offset` | 1/cos(45°) ≈ 1.4142 |

> 與 Type 39 相同的三角函數架構，offset 值依 Member 不同。

**格式**: `42-{A}-{MEMBER}-{H} {FIG}[-{E修正}]`

| 段位 | 意義 | 說明 |
|:---:|------|------|
| 1 | Type 編號 | 固定 `42` |
| 2 | 管徑代碼 | LINE SIZE "A"（如 2B, 4B, 8B） |

## 與 Type 39 的關鍵差異

| 項目 | Type 39 | Type 42 |
|------|---------|---------|
| 接觸對象 | 直接 Pipe | **Trunnion（短管支座）** |
| 力傳遞 | Pipe → Lug → Vessel | **Pipe → Trunnion → Vessel** |
| H 公式 | `K - R - t - E - 30`（線性） | **`F - √(R² - E²)`（圓弧）** |
| Lug Plate | M-34 / M-35 / M-36 | **無**（用 Ø22 + 3/4"×50 M.B. 直接接合） |
| 可調整性 | 少 | **多（C/S Shim 現場調整）** |
| Trunnion | 無 | **有（依管徑查表）** |
| 管徑資訊 | 無 | **有（LINE SIZE → E → Trunnion）** |
| 複雜度 | 中 | **高** |

## 運算邏輯

```
1. 輸入: F (Vessel距離), R (Vessel半徑), LINE SIZE "A"
2. 查表2: LINE SIZE → E, TRUNNION SIZE "B"
3. 反算 H = F - √(R² - E²)
4. 檢查 H ≤ H_MAX (依 member 查表1)
5. 選擇 θ (30° 或 45°) → FIG-A 或 FIG-B
6. 查表1: G = 1.155H+offset (30°) 或 1.414H+offset (45°)
7. BOM:
   - Member ×1 (長度=G)
   - L75X75X9 支撐角鋼 ×1
   - Trunnion ×1 (size B)
   - C/S Shim (現場)
   - 3/4"×50 M.B. set
```
