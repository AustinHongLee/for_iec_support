# M-35 — Lug Plate TYPE-D（45° 轉接板）

> **圖號**: M-35 &nbsp;|&nbsp; **分類**: 零件 (Component) &nbsp;|&nbsp; **狀態**: 🔲 參考
> **圖面編號**: E1906-DSP-500-006 &nbsp;|&nbsp; **日期**: 12/12/19

45° 角度 Lug Plate，用於斜撐與 Vessel/結構間的載荷轉接。
主要搭配 θ=45° 支撐系統（如 Type 39 FIG-B 的 DETAIL Z）。

---

## 結構示意圖

```
         ┌──── A ────┐
    t    │            │
  ──┤    │   ╱ C ╲    │
    │    │ ╱ D ╲╱ D╲  │
    │    │╱  ⊙  ⊙  ╲  │──── MEMBER
    B    │ ╲ E ╱╲ E╱  │
    │    │   ╲  ╱  ╱   │
    │    │    ╲╱  ╱    │
    │   45°   ╲ ╱     │  ← 板角度 45°
  ──┤    │     ╲      │
         │  ØJ HOLE   │  "K" BOLT
         │  FOR BOLT  │
    T    │            │
  PLATE  └─ "T" TH'K ─┘
         INSULATION
```

**力傳遞**：PIPE → INSULATION → LUG PLATE (45°) → K BOLT → MEMBER

---

## 本質與定位

Lug Plate 家族共三型：

| 型號 | 角度 | 用途 |
|:---:|:---:|------|
| TYPE-C (M-34) | 0° (平面) | 水平接、一般固定 |
| **TYPE-D (M-35)** | **45°** | **一般斜接轉換** |
| TYPE-E (M-36) | 60° | 30° 角度補償（含 0.577T 修正） |

> **核心功能**：把斜撐幾何 (45°) 轉換成可鎖螺栓的平面接口。

## 工程 NOTES

1. **DESIGNATION**: `LGP-D-{n}` — D 表示 TYPE NO.，LGP 表示 LUG PLATE
2. **MATERIAL**: Same as the metal to which it is connected（與連接構件同材質）
3. **REINFORCING PAD**: 若加補強墊板，PAD 厚度已含在 A 或 A+t 內
