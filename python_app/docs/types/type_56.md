## 系統本質

TYPE-56 是 51~56 系列中**唯一自成一體的結構式管線檔止**——不引用 D-80（TYPE-66）也不引用 D-81（TYPE-67），而是自行定義結構構件（MEMBER "C"）、尺寸表和鞍座接觸面。

核心特徵：
1. **MEMBER "C"** 為主承載構件——由 H 型鋼切割或鋼板製造，尺寸按管徑直接查表
2. **3mm gap 側板** 提供橫向擋止——與 TYPE-52~55 相同的 3mm 間隙構造
3. **無 Clamp、無 Gasket、無 Guide** ——純鋼構焊接系統
4. **Designation 極簡**：`56-{size}B`——全系列最簡單，無 HOPS/LOPS/TABLE A/B

```
TYPE-51~55: 引用 D-80 或 D-81 的 Shoe 系統（標準零件組合）
TYPE-56:    自成一體的結構式檔止（自行定義 Member "C" + 尺寸）
```

> **TYPE-52~55 是「用標準件組合控制管線」**
> **TYPE-56 是「用結構鋼直接擋住管線」**

## 系統定位

### 51~56 系列三分支架構

```
自成一體分支:
├── TYPE-51: 自有 80° 鞍座（自由承托，無橫向約束）
└── TYPE-56: 自有結構框架（檔止，有橫向約束）

D-80 分支（無隔離）:
├── TYPE-52: Shoe + Retainer（防偏）
└── TYPE-53: Shoe + Guide（導向）

D-81 分支（隔離型，AS/SS 專用）:
├── TYPE-54: Shoe + Clamp + Gasket（隔離固定）
└── TYPE-55: Shoe + Clamp + Gasket + Guide（隔離固定+導向）
```

### TYPE-56 vs TYPE-52 比較

兩者都提供橫向約束（3mm gap），但設計思路完全不同：

| 比較 | TYPE-52 | **TYPE-56** |
|------|------|------|
| Shoe 來源 | D-80（TYPE-66） | **自有 MEMBER "C"** |
| 構件定義 | 查 D-80 表 + L-angle | **自有尺寸表（A/B/C/D/E/R）** |
| 側邊構件 | L40×40×5 retainer | **結構側板（框架一體）** |
| Rein. Pad | D-80 系統 | **D-91**（不同參照！） |
| Designation | 長（含 HOPS/LOPS/TABLE A/B） | **極簡 `56-{size}B`** |
| 管徑範圍 | 1/2"~24" | **3/4"~42"** |
| 設計本質 | 標準件組合 | **客製結構框架** |

## 連接方式

### 全焊接系統

TYPE-56 是 51~56 系列中**唯一完全無機械連接的型式**——所有連接全部焊接：

| 連接界面 | 方式 |
|------|------|
| Pipe ↔ 鞍座/平板 | 接觸（重力壓持） |
| 鞍座 ↔ MEMBER "C" | 焊接 |
| MEMBER "C" ↔ 側板 | 焊接 |
| Frame ↔ Base Beam | 焊接 |

> 無 bolt、無 clamp、無 gasket。

### 焊接規格

| 管徑範圍 | 焊接 |
|:---:|------|
| 3/4"~24" | 6V TYP. |
| 26"~42" | 6V TYP., 3 SIDES TYP. |

### 荷重路徑

```
Pipe（水平管線）
→ 鞍座弧面 / 平板（接觸承托）
→ MEMBER "C"（主承載構件）
→ 側板 / 加勁板（結構框架）
→ Base Beam
```

## 約束行為

| 自由度 | 狀態 | 說明 |
|:---:|:---:|------|
| 垂直（↓） | **SUPPORTED** | 鞍座/平板承重 |
| 垂直（↑） | **FREE** | 無 clamp |
| 軸向（Axial） | **見下方分析** | 依尺寸而異 |
| 橫向（Lateral） | **RESTRAINED** | 3mm gap 側板，與 TYPE-52 相同構造 |
| 旋轉 | **LIMITED** | 鞍座弧面約束（3"+） |

### 軸向行為分析

圖面上 **3mm 間隙** 明確標示在橫向（側向），但軸向方向的約束狀態需分析：

- **≤2-1/2"**：管線坐在 PL 100×100×6 上方，可沿軸向滑動 → **FREE**
- **3"+**：管線坐在 MEMBER "C" 鞍座中，鞍座弧面提供部分軸向摩擦 → **PARTIALLY FREE**
- NOTE 2 使用 "**STOPS**" 一詞，暗示本型在工程系統中被歸類為管線檔止

> ⚠️ 與 TYPE-54/55 的 FIXED（Clamp 夾固）不同，TYPE-56 的約束來自結構框架本身，
> 不依賴機械連接件。橫向約束明確；軸向約束程度依安裝細節和實際框架構型而異。

### 51~56 系列約束比較

| 自由度 | TYPE-51 | TYPE-52 | TYPE-53 | TYPE-54 | TYPE-55 | **TYPE-56** |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| Axial | FREE | PARTIAL | FREE | FIXED | FIXED | **PARTIAL** |
| Lateral | FREE | RESTRAINED | STRONGLY RESTRAINED | FIXED | FIXED | **RESTRAINED** |
| Vert ↑ | FREE | FREE | FREE | FIXED | FIXED | **FREE** |
| 隔離 | ❌ | ❌ | ❌ | ✅ | ✅ | **❌** |
| 定位 | 自由承托 | 防偏 | 導向 | 隔離固定 | 隔離固定+導向 | **結構檔止** |

## 設計流程

```
1. 確定: LINE SIZE (3/4"~42")
2. 選結構型態:
   - ≤2-1/2" → 標準 PL 100×100×6
   - 3"~24" → 查表 1 (D-67): A/B/C/D/E/R
   - 26"~42" → 查表 2 (D-67A): A/B/C/D/E/R + REIN. PAD (D-91)
3. 查 R 值（= OD/2，確認鞍座弧面尺寸）
4. 確認 MEMBER "C" 來源（H 型鋼 / 鋼板）
5. 確認側板/加勁板厚度 E
6. 編碼: 56-{size}B（極簡）
```

> 與 TYPE-52~55 的設計流程差異：
> - 不需查 D-80/D-81 Shoe 表
> - 不需查 D-80A TABLE A/B
> - 不需考慮保溫厚度 / 材質符號
> - 不需計算 HOPS / LOPS
