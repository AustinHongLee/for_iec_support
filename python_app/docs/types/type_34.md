## 系統本質

L型結構 — 上橫梁(L, 承管)+右側立柱(H), 管在上梁, 無M-42

與Type 33差異: 33=管在下梁(框撐管), 34=管在上梁(梁撐管)。無螺栓, 全焊接。

## 設計重點

- 本型式不使用 `M-42`，也沒有額外下部構件或螺栓。
- 結構本質為「右側單柱 + 上橫梁」的 L 型懸臂；左側固定端由 `EXISTING STEEL` 提供。
