# Type 37 — 斜撐懸臂支撐 (Braced Cantilever Support)

> **圖號**: 37 &nbsp;|&nbsp; **分類**: 支撐型式 &nbsp;|&nbsp; **狀態**: ✅ 已實作

焊接於 EXISTING SURFACE 的斜撐懸臂，上主梁承管 + 斜撐傳力至牆面。
A 型 (θ=30°) 較水平，B 型 (θ=45°) 較陡峻。全焊接，無螺栓。

## 編碼格式

**格式**: `37-{MEMBER}-{H}{FIG}[-{C}]`

| 段位 | 意義 | 說明 |
|:---:|------|------|
| 1 | Type 編號 | 固定 `37` |
| 2 | 型鋼代碼 | L75, C100, C125, H100, H125, H150 |
| 3 | H + FIG | 數字=H(mm**直接**)，末位=A/B |
| 4 | C 尺寸 (選填) | ×100mm，預設 `02` (200mm) |

**範例**:
- `37-C125-1200A` → C125, H=1200mm, A型(30°), C=200mm(預設)
- `37-C125-1200B-05` → C125, H=1200mm, B型(45°), C=500mm

> ⚠️ H 是 **直接 mm**（如 1200），不是 ×100！與 Type 31~34 的 LLHH 格式不同。

## 運算邏輯 — 斜撐長度

### 通用公式

斜撐分為兩段：**穿越型鋼區** (d 段) + **自由跨距區** (H 段)

$$L = \frac{d}{2 \cdot \cos\theta_Y \cdot \cos\theta_X} + \frac{H}{\cos\theta_X}$$

其中 `d` = member 深度 (C125 → 125mm)

### A 型 (θ=30°, θ_Y=60°)

因 cos(60°)=0.5，化簡為：

$$L_A = \frac{d + H}{\cos 30°} = (d + H) \times \frac{2}{\sqrt{3}}$$

### B 型 (θ=45°, θ_Y=45°)

因 cos²(45°)=0.5，d 和 H 使用**不同**比例因子：

$$L_B = d + H\sqrt{2}$$

### VBA 4-Step 忠實還原

```python
# Step 1: 半截面在 θ_Y 方向的投影
first_step = (d / 2) * tan(θ_Y)

# Step 2: 半截面的斜撐長度
second_step = round(√(half² + first_step²))

# Step 3: 型鋼區斜撐（含 θ_X 投影）
third_step = round(√((second_step × tan(θ_X))² + second_step²))

# Step 4: 自由區斜撐
forth_step = round(√((H × tan(θ_X))² + H²))

# 總長
L = round(third_step + forth_step)
```

### 驗算範例 (C125-1200A)

| 項目 | 值 |
|------|-----|
| d (member 深度) | 125 mm |
| H (懸臂長) | 1200 mm |
| C (預設) | 200 mm |
| 上主梁 | 1200 + 200 = **1400 mm** |
| 斜撐 (A型) | (125+1200)/cos30° = **1530 mm** |
| VBA 4-step 結果 | **1530 mm** (0mm 誤差) |

## 備註

- 無 M-42 底板（焊接於既有表面）
- 無螺栓，全焊接 (6mm fillet weld)
- C 延伸段預設 200mm，可透過第四段自訂
- H150 規格特殊：使用 7mm 翼板 (非預設 10mm)
- 邊界行為：d=0 時退化為純三角函數；H=0 時僅剩型鋼穿越段
