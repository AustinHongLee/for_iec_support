## 系統本質

雙管共用托板，角鐵立柱，螺栓固定(M-37 Lug Plate不含供貨)

雙管平行坐在橫向托板上的立柱支撐。Member C(L50/L65/L75)垂直+水平段。H/L現場裁切，H≤1500mm，L≤1000mm。A/B可不對稱。底部用M-37 Lug Plate螺栓固定(不列入材料)。無U-bolt、無M-42。
