# M-47 — Compressed Gasket（非石棉墊片）

> **圖號**: M-47 &nbsp;|&nbsp; **分類**: 零件 (Component) &nbsp;|&nbsp; **狀態**: ⚠️ 部分轉錄
> **圖面編號**: E1906-DSP-500-006

M-47 是管線與支撐構件之間的隔離墊片，主要用在不宜直接接觸碳鋼支撐的場景，例如 Type 13、Type 54、Type 55、Type 67。

## 厚度來源說明

目前 repo 內對 M-47 厚度有兩條穩定線索：

- `<= 24"`：沿用既有 `Type 13` calculator 的 `3t` 假設
- `26"~42"`：依 `Type 67 / D-81A` 文件明確提到 `NON-ASBESTOS GASKET, 1.5t`

因此目前 `m47_table.py` 採：

- `<= 24"` → `3.0t`
- `26"~42"` → `1.5t`

> 這代表 M-47 已經有可用的尺寸 lookup，但還不是逐字逐欄位的完整 PDF 轉錄。
